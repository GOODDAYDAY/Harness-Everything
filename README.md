# Harness-Everything
Harness-Everything is a provider-agnostic AI coding harness. Point it at any Anthropic-compatible LLM (Claude, DeepSeek, etc.) and a codebase, and it will autonomously plan, execute, and evaluate improvements — either as a single task or as a multi-phase pipeline with three-way planning, dual evaluation, and cross-round memory.
